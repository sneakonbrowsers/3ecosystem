Plugin install callback tester

Enter any plugin URL (HTTP, localhost, file) and click one of the test buttons to trigger the install callback Sneakon handles.


HTTPS format:
https://sneakonbrowsers.github.io/callback/install?plugin=<url>

consurf format:
consurf://install?plugin=<url>
